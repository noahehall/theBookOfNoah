## categories

- headers generally belong to a few use cases, and primarily to one

## request headers

- accept
- accept-encoding: types of compression algorithm the UA understands
- accept-language
- connection
- cookie
- host
- referer
- user-agent

## response headers

- accept-ranges
- cache-control
- content-encoding: if the body is compressed, then the algorithm used
- content-length
- content-type
- last-modified
- status
- vary
- server
- date
